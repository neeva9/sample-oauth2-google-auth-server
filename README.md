# Demo for Shopping Cart

#### Prerequisite:
````
Java 8
IntelliJ IDEA
Gradle
````

#### Database:
````
H2 embedded connection
````

#### Build:
````
gradle clean build
````

#### Run Application:
````
Use the below JVM argument:
-Dspring.profile.actove=h2
````

#### Swagger:
````
http://localhost:8080/swagger-ui/index.html#/ 
````
